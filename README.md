## Projects

### [ml-pipeline-deployment](https://github.com/cymmerman/ml-pipeline-deployment)
A production-style REST API for turbofan engine anomaly detection. FastAPI backend with SHAP explainability, KS-test drift detection, a WebSocket-based engine simulator that replays NASA C-MAPSS data with fault injection, SQLite audit logging, and a live monitoring dashboard with what-if analysis and side-by-side engine comparison. The part I'm most proud of is the simulator — you can inject bearing wear or compressor stall mid-stream and watch the model react in real time.

**Stack:** FastAPI · scikit-learn · SHAP · pandas · aiosqlite · Docker · Chart.js

---

### [turbofan-predictive-maintenance](https://github.com/cymmerman/turbofan-predictive-maintenance)
Predictive maintenance pipeline on the same NASA C-MAPSS dataset. Isolation Forest for anomaly detection, Random Forest for remaining useful life prediction (R² ~0.99), SHAP explainability, fleet health scoring, and an interactive browser dashboard. This is where the model used in the deployment project was originally developed.

**Stack:** scikit-learn · SHAP · pandas · numpy · matplotlib

---

### [mcmc-bayesian-flight-estimation](https://github.com/cymmerman/mcmc-bayesian-flight-estimation)
Metropolis-Hastings MCMC implemented from scratch for Bayesian uncertainty quantification of missile flight parameters. Sub-1% error on parameter recovery from noisy sensor data. No sampling libraries — just numpy and the math.

**Stack:** numpy · scipy · matplotlib

---

### [aerospace-nlp-document-classifier](https://github.com/cymmerman/aerospace-nlp-document-classifier)
NLP classification and search system for aerospace documents. TF-IDF with bigrams, Naive Bayes and SVM classifiers (~80% accuracy on unseen docs), cosine similarity search, word clouds, and an interactive HTML search interface.

**Stack:** scikit-learn · NLTK · pandas · numpy
